# 教案：电容器的电容

## 教学目标

### 知识与技能

1. 理解电容器的定义——相互靠近且彼此绝缘的两个导体构成电容器
2. 能描述电容器的充电和放电过程，知道电子转移方向，体会电荷守恒
3. 理解电容的物理意义，掌握定义式 $C = Q/U$ 及单位
4. 知道平行板电容器的电容由 $S$、$d$、$\varepsilon_r$ 决定，了解决定式 $C = \varepsilon_r S/(4\pi k d)$
5. 了解额定工作电压与击穿电压的区别

### 过程与方法

1. 通过发散举例体会电容器定义的广泛适用性
2. 通过电路模拟实验，观察充放电过程中电子流动与 $I$-$t$ 变化
3. 通过电荷共享实验，探究 $Q$ 与 $U$ 的定量关系，经历观察、归纳、建立概念的探究过程
4. 通过水桶类比，建立电容的直观物理图像，体会类比法的价值
5. 通过平行板电容器模拟实验，经历猜想、控制变量、验证的科学探究过程

### 情感态度与价值观

1. 体会“储存电荷”的工程需求背后的物理思想
2. 深化对电荷守恒定律的认识
3. 通过实验探究和类比思维激发学习兴趣

## 教学重点

1. 电容器的定义及充放电过程的微观解释
2. 电容的概念——$C = Q/U$ 的物理意义
3. 平行板电容器电容的决定因素

## 教学难点

1. 电容器保持电荷的原理（静电引力与绝缘条件）
2. 电容概念的抽象过程——从 $Q/U$ 比值到“储存本领”的物理理解
3. 区分电容的定义式与决定式

## 教学过程

### 一、引入：如何储存电荷？

回顾三种起电方式，指出它们只能让物体带电、不能储存，引出电容器课题。

### 二、电容器的定义与常见电容器

给出定义“相互靠近且彼此绝缘的两个导体”，阐述两个条件。发散举例（地球与火星、人与金属板、云层与大地）说明定义普适性，其中云层与大地为后续击穿电压埋伏笔。实物照片展示三类常见电容器，强调本质均为两个靠近且绝缘的导体。

### 三、充放电过程

教师模拟充放电电路实验，演示单刀双掷开关切换充、放电，观察电流变化和极板带电情况。归纳电子转移方向和电势差变化：充电时与正极相连的极板失去电子带正电，另一极板获得电子带负电，电势差升至电源电压；放电时电子从负极回流正极，电荷中和，电势差降低。强调电荷守恒——电荷只是转移，不创造不消灭。

板书画出 $I$-$t$ 曲线，引导学生分析：由 $I = Q/t$ 知曲线下面积对应电荷量的变化；放电过程中外负载电阻一定，由 $I = U/R$ 知 $U$-$t$ 曲线与 $I$-$t$ 曲线变化趋势相同——电势差从电源电压降至零。曲线特征是先快后慢：起始时电势差大、电流大、电荷转移快；随着电势差减小，电流减小，变化趋缓。

### 四、为什么“相互靠近”是必要条件？

设计思想实验（负极接地）拷问定义合理性。学生直觉“接地即流失”，但静电引力使正极板电荷被拉住、几乎不流入大地——只要不构成闭合回路，电荷得以保留。由此证明“相互靠近”不是随意规定，而是保证静电引力束缚电荷的必然要求，与“彼此绝缘”共同构成储存电荷的充要条件。

### 五、电容器的带电量与能量转化

电容器两极板总是带等量异种电荷，总电荷为零不能反映储存量，故带电量取一个极板电荷的绝对值。通过充放电过程引入能量转化：充电时电源能量存入电容器，放电时电容器能量释放到电路——电容器同时储存电荷和能量。

### 六、探究储存电荷的本领——建立电容概念

类比质量、电阻引入“本领”概念，提问“不同电容器的储存本领是否一样？如何量化？”，引出定量探究需求。

教师模拟电荷共享实验电路（教材图10.4-4）：同一电容器 $Q$ 与 $U$ 的比值是定值，不同电容器该比值不同——$Q/U$ 反映储存本领。

水桶类比：$S = V/h$ 对应 $C = Q/U$，说明为什么是 $Q/U$ 而非 $U/Q$。配合同步动画建立直观图像。

### 七、电容的定义与课堂练习

三步递进：定义（$Q$ 与 $U$ 的比值，反映储存本领）、定义式（$C = Q/U$）、单位（法拉 F，常用 $\mu$F、pF）。实物传看印证 1F 之巨大，并自然注意到工作电压标注，过渡至下一问题。课堂计算巩固定义式使用。

### 八、工作电压与击穿电压

聊天气泡一问一答引入，从闪电（云层与大地电容器被击穿）引出击穿概念。归纳额定工作电压与击穿电压的区别：额定电压为长期可靠工作的上限，击穿电压为介质承受的极限，超出则不可逆损坏。额定电压须低于击穿电压，留安全裕度。

### 九、探究平行板电容器的电容

转折提问“电容的大小和什么因素有关？”，小组讨论提出猜想（$S$、$d$、介质）。

教师模拟平行板电容器实验，控制变量法逐一验证：$S$ 增大则 $C$ 增大，$d$ 增大则 $C$ 减小，$\varepsilon_r$ 越大 $C$ 越大，$U$ 变化则 $Q$ 等比例变化但 $C$ 不变。结论：$C$ 由结构参数决定，与 $U$ 无关。给出决定式 $C = \varepsilon_r S/(4\pi k d)$，强调与定义式 $C = Q/U$（量度式）的区别。

### 十、课堂小结

四块回顾：电容器定义、电容定义式与单位、平行板电容器决定式、工作电压注意。

### 十一、课后作业

两类动态分析：$U$ 不变（接电源）与 $Q$ 不变（断电源）情形下，$S$、$d$、$\varepsilon_r$ 变化对 $C$、$Q$（或 $U$）、$E$ 的影响，表格呈现，为习题课铺垫。
